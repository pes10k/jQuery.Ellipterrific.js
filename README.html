jQuery Ellipterrific ReadMe

About Ellipterrific

Ellipterrific is a simple jQuery plugin to trim text in an element to fill a given space and make it end with …

This plugin came out of an effort to get the CSS3 text-overflow: property to act the way I, expected it to, which is to not require all text to be on a single line before clipping and adding the ellipse.

There are lots of other fine jQuery plugins for doing similar things. I wrote this one because the other plugin's I tried either required a lot of additional markup in the HTML, we're slow and made the resizing process visible to the user, or otherwise didn't quite fit my needs. It was also a good way to learn more about GitHub.

Using Ellipterriffic

Using Ellipterriffic is simple. Its called like any other jQuery plugin by selecting a group of DOM elements and calling ellipterrific() on them. No special mark up is needed.

Note: Ellipterriffic converts all content to text when calculating sizes, so any formatting will be not be preserved.

To use, first size your element with the maximum size the text should take up in CSS (ex height, width), and set it to have css style property overflow to be hidden. This also makes sure that we degrade smoothly, since non-js visitors will see the element laid out correctly, but with the text clipped.

Configuring Ellipterrific

Ellipterrific accepts the following configuration options:

ellipsis_string (string)
The character that will be appended to the end of the element's text to note that some text has been truncated. Can be a valid HTML string or plain text. Defaults to "…".
split_on_words (bool)
Whether to split the element's text on words or individual characters. Defaults to false.
